PromptBench

The open benchmark for AI-native software engineering.

PromptBench is an open-source technical interview platform for the AI-native engineer.

Practice real-world engineering challenges using AI coding agents like Claude Code, Codex, Gemini CLI, OpenCode, and Cursor while PromptBench records your prompts, tracks iterations, evaluates outcomes, and generates replayable scorecards.

Built for developers, hiring teams, and educators who want to understand and improve AI-native engineering skills.

⸻

Why PromptBench?

Traditional technical interviews measure how well someone writes code.

Modern software development increasingly involves collaborating with AI systems to transform requirements into working software.

PromptBench measures the skills that matter in this new workflow:

* Writing clear specifications
* Managing context effectively
* Iterating on solutions
* Debugging with AI assistance
* Delivering working software
* Communicating intent and constraints

PromptBench is designed to evaluate outcomes, not prompt tricks.

⸻

How It Works

1. Select a challenge
2. Connect your preferred AI coding agent
3. Solve the challenge using prompts
4. PromptBench records the session
5. Tests and rubrics run automatically
6. Receive a scorecard and replay

npx promptbench start
